# Focus Todo CLI

A minimalist command-line todo application that helps you stay focused by showing only one task at a time.

## Overview

This todo app follows a unique approach to task management: it only reveals your next todo after you've completed the current one. This design helps eliminate the overwhelming feeling of seeing all pending tasks and encourages focused, sequential task completion.

## Features

- Single task focus: see only your next todo
- Simple CLI interface
- Add new todos
- Mark todos as complete
- Sequential task revelation
- Distraction-free task management

## Why One Task at a Time?

Traditional todo lists can be overwhelming and lead to decision paralysis. By showing only one task at a time, this app helps you:

- Stay focused on the current task
- Reduce anxiety about pending tasks
- Increase productivity

## Setup

1. Clone the repository
2. Navigate to src directory
3. Run `go run .`
